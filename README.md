This repository demonstrates automation of a website using the Page Object Model (POM), following best practices and a maintainable test design. The goal of this framework is to provide fast, reliable, and scalable UI automation, ideal for CI/CD pipelines and enterprise-level projects.

✨ Key Highlights

✔ Full-Scale Test Automation Framework
✔ Built with Playwright + TypeScript
✔ Page Object Model (POM) for maintainable code structure
✔ Cross-browser testing (Chromium, Firefox, WebKit)
✔ Parallel test execution
✔ Built-in Playwright reporting
✔ Easy CI/CD integration
✔ Clean and scalable project architecture

🧠 Framework Architecture

The framework follows the Page Object Model (POM) design pattern to improve:
✔ Code reusability
✔ Maintainability
✔ Test readability
✔ Scalability for large projects

project-root
│
├── tests/               # Test specifications

├── fixtures/            # Custom fixtures
├── utils/               # Utility functions
├── test-data/           # Test data management
├── playwright.config.ts # Playwright configuration
└── README.md
